# Weather APP Using Python
This is a simple python project that fetches real-time weather data using openweather API
## Features
-Get weather by city name
-Shows temparature in Celsius
-Shows weather condtion 
## Technologies Used
- Python 
- Requests library
- REST API 
## How To Run
1. Install requests:
   pip install requests
2. Run:
   python main.py
